游戏名：《信长之野望：新生》

描述：3月28日在光荣特库摩40周年纪念直播中，官方发布信长系列第16作《信长之野望：新生》宣传片，目前游戏正在开发中，预计将在年内发售。官方表示游戏详情将在今后逐步公开。
游戏预价￥129

游戏名：《毁灭全人类 Destroy All Humans》

描述：重制版的《毁灭全人类》,游戏风格诙谐，背景设置在20世纪50年代。玩家将扮演一位名为Crypto-137的外星入侵者，可以伪装成人类，使用超能力和各种奇特的武器。重制版游戏将于2020年登陆PS4，Xbox One及PC平台。
游戏预价￥89

游戏名：《英灵神殿》

描述：MOD制作者“aC0C0NUT”近日发布了一个全新的《英灵神殿》的高清材质包，这个材质包重做了游戏中的255处贴图。如果你是一位追求画质的《英灵神殿》玩家，那么这个材质包绝对是必备之物。
游戏预价￥299

游戏名：《黑暗之魂3》

描述：《黑暗之魂3》是一款黑暗奇幻题材的第三人称硬核动作RPG游戏。这款作品是《黑暗之魂》系列的第三部正统作品，这款游戏由于其超高的难度和出色的游戏性，受到了玩家们的欢迎。
游戏预价￥289

游戏名：《怪猎崛起》

描述：日前卡普空旗下名作新篇《怪物猎人：崛起》正式发售，有玩家反映因为追加姿势包导致游戏崩溃无法启动，3月28日卡普空发布致歉公告。
游戏预价￥119

游戏名：《二次灭绝》

描述：合作FPS游戏《二次灭绝》几个月之前登陆了Steam抢先体验，近日官方宣布本作将于4月28日登陆PC和主机端的Xbox Game Pass
游戏预价￥129

游戏名：《刺客信条:英灵殿》

描述：育碧发布了一个热修复补丁，对《刺客信条：英灵殿》中可能会导致崩溃问题的部分内容进行了紧急移除。本作开发团队终于发布了最新升级档，解决了崩溃问题，并恢复了被移除的内容。
游戏预价￥299

游戏名：《对马岛之魂》

描述：去年大热的《对马岛之魂》官方近日宣布，该作将被改编成好莱坞真人电影，不少玩家认为《对马岛之鬼》“境井仁”扮演者其实可以出演真人电影《底特律：变人》中“康纳”扮演者Bryan Dechart也大家一样在推特上直接号召大家支持。
游戏预价￥339

游戏名：《古惑狼4：时机已到》

描述：目前，《古惑狼4：时机已到》现已登陆PC台服战网，需全程联网游玩，游戏支持4K，每秒帧数无上限，购买《古惑狼4》并完成游戏内第二个关卡的玩家，还可以免费获得Crash和Coco的骸骨造型。
游戏预价￥295

游戏名：《战地6》

描述：据舅舅党爆料，EA和DICE最近为《战地6》发布了一个问卷调查。显然EA正在测量玩家对一些功能的兴趣，有可能会被加入到游戏中。
游戏预价￥339

游戏名：《赛博朋克2077》

描述：国外网友“PricklyAssassin”发现，《赛博朋克》Epic版本服务器容量最近从100GB扩展到了500GB，一般来说这种事预示着游戏未来几个月会推出新内容。此外它的“附加内容”区域也发生了改变
游戏预价￥399

游戏名：《Moonglow Bay》

描述：Bunnyhug开发、发行的钓鱼RPG新作《Moonglow Bay》已经上架Steam，并预定将于2021年内发售，该作将登陆Xbox Series/One和PC（游戏将加入XGP），该作支持简中。
